# vuedome

> A Vue.js project

## Build Setup

``` bash

# 克隆项目
git clone https://github.com/MobileTemplate/vuedome.git

# 安装依赖关系
npm install

# 运行项目  localhost:8080
npm run dev

# 项目打包
npm run build

```

For detailed explanation on how things work, consult the [docs for vue-loader](http://vuejs.github.io/vue-loader).
